Hybrid execution

Agent mode
Mode

Agentic planning sketches a plan first, checks itself for gaps, and verifies its own answer once before showing it. Web search (toggle above) is a native live search on OpenAI-family models. Tools give Coctus its own real web_search + web_fetch + wikipedia + weather + image_gen, plus JS/Python code execution in a sandbox — these work on any model. Run to completion keeps Coctus working until a multi-part task is actually finished, instead of stopping partway. Show thinking asks the model to reason out loud first, in a separate collapsible panel above the answer — a visible reasoning trace, not the model provider's own internal chain-of-thought.

Special modes

Deep Research forces tools + planning on, allows more search rounds, and requires multiple cross-checked sources plus a Sources section. Team mode splits the work across three sequential agents — Researcher → Writer → Critic — visible in the Trace tab as they hand off. The two are mutually exclusive; turning one on turns the other off.

Second opinion optional

When set, a second, different model independently reviews the draft answer for mistakes the first model might share a blind spot on. Costs one extra model call per response.
